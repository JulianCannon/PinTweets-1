"Facebook feels like a mall. Twitter feels like the street."
-William Gibson

As succinct as this comment is, in one important aspect it is not true. On the street, geography is paramount. On Twitter, it is a little line in a user profile.

This website seeks to restore geography's place in the wonderful absurdity that is Twitter. Unfortunately, it does not work (yet) in Microsoft Internet Explorer.

It does work in Google Chrome (recommended), Mozilla Firefox, and Apple's Safari. Please use one of these website and enjoy the site.
